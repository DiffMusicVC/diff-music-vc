Demo: Adapting Diffusion-Based Music Synthesis to Speech and Singing Voice Conversion

How to View the Demo:

1. Extract all contents of the provided DiffMusicVC.zip file to a folder of your choice
2. Open the folder containing the extracted files
3. Locate the DEMO.html file and open it using a modern web browser such as Google Chrome, Mozilla Firefox, or Microsoft Edge (tested on all three browsers). For best visibility, it is recommended to use one of these browsers

You can open the DEMO.html file by:

1. Double-clicking it
2. Right-clicking the file, selecting "Open with," and choosing your preferred browser
